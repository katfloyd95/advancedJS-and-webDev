  1. What are libraries?

    2. The same way we import files from out directory such as index.js etc., we can do the same with files from the internet. This can be done in two different ways:

    • Download the library files and throw them into your directory with your index.html file

  2. What is a CDN?

    3. A server that hosts the file you want to use somwehere, to which you point your script tag
