    NOTE: The '$' in your Shopify theme files is JQUERY.

    The '$' replaces 'document.querySelector', but it is not EXACTLY the same. The 'document.querySelector' create DOM element, whereas the '$' creates a jQuery HTML element.

    This essentially means that under the hood, they each have a different set of methods. Don't use DOM methods on a jQuery element, and don't use jQuery methods on a DOM element

    Don't use jQuery too much, rather just follow the patterns of the theme code
